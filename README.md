# Phishing URL Detection System

An AI-based phishing URL detection system using Java and Machine Learning.

## Features
- Detects phishing and legitimate URLs
- Machine learning model integration (Weka)
- Feature extraction from URLs
- Java Swing UI

## Tech Stack
- Java
- Machine Learning
- Weka
- Cybersecurity

## How to Run
1. Clone the repository
2. Open in VS Code or IntelliJ
3. Run the Java application
4. Enter a URL to check
